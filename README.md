# 🛡️ Clan Event Bot

Мощный Telegram-бот для организации клановых ивентов, сбора участников и контроля дисциплины. Бот автоматически формирует списки, управляет таймерами и требует отчеты от неявившихся игроков.

## ✨ Основные возможности

- ⏱️ **Гибкий таймер:** Сбор участников на заданное время (по умолчанию 5 минут).
- 👑 **Защита прав:** Управлять сбором могут только администраторы группы.
- 🔔 **Умные уведомления:** Красивые списки участников с активными ссылками на профили.
- ⚠️ **Контроль дисциплины:** Автоматическое предупреждение о необходимости отчета перед администрацией в случае неявки.
- 📝 **Легкое использование:** Интерфейс через кнопки и простые команды.
